# Echo Web Works

Fast, modern, beautifully designed websites for ambitious brands.

## Portfolio sites

| Site | Folder |
|------|--------|
| Aquatica (swim coaching) | `portfolio/aquatica/` |
| Cleanic (cleaning service) | `portfolio/cleanic/` |
| Nails salon | `portfolio/nails/` |
| Signature Detailing (cars) | `portfolio/cars/` |

## Structure

```
index.html          ← Agency landing page
script.js           ← Shared JS (theme toggle, scroll animations)
styles.css          ← Shared CSS
portfolio/
  portfolio.html    ← Portfolio gallery page
  aquatica/
  cleanic/
  nails/
  cars/
```

## Running locally

Open `index.html` in a browser — no build step required. All sites are plain HTML/CSS/JS.
